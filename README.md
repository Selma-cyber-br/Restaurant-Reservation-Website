# Restaurant Reservation Website

## Overview

A modern restaurant website featuring menu presentation, reservation functionality, responsive design, and a user-friendly interface.

The project was developed to simulate a real-world restaurant booking experience while focusing on frontend development best practices.

---

## Features

* Online reservation form
* Interactive user interface
* Responsive design
* Mobile-friendly navigation
* Menu presentation
* Contact section
* Smooth user experience

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Objectives

* Improve frontend development skills
* Practice responsive design techniques
* Build a realistic business-oriented website
* Enhance user experience design
